mic drops is a musical toy.

talk to it, sing to it, beep at it - it will respond.

notes:

  • giving the browser microphone permission is required.
  • the we listen control adjusts the mic sensitivity. try turning it up if you're not getting a response or down if it's all too much.
  • this currently works best on chrome desktop at the moment.
  • use headphones to avoid feedback. or embrace it.

see the code at github or glitch.
